6、python第三方软件包安装

a、安装pip

更换国内源：pip config set global.index-url https://mirrors.aliyun.com/pypi/simple

b、或者直接用pacman安装即可，eg：

sudo pacman -S python-numpy python-scikit-learn python-pandas python-matplotlib

7、在Linux中自定义”pause”命令

效果和windows下的system("pause")一样。

sudo vim /usr/bin/pause

sudo chmod 0755 /usr/bin/pause

8、docker

安装：sudo pacman -S docker

配置阿里云镜像服务：https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

sudo vim /etc/docker/daemon.json

sudo systemctl daemon-reload

sudo systemctl restart docker

9、开启docker中mysql服务(docker相关命令自行百度)

sudo docker pull mysql:5.7.29

sudo docker run --name mysql57 -p 3307:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7.29

若idea等其他客户端无法连接docker中的mysql，可以参考链接1、链接2

11、开启docker中elasticsearch服务

sudo docker pull elasticsearch:6.8.8

sudo docker network create esnet：如果需要安装kibana等其他，需要创建一个网络，名字任意取，让他们在同一个网络，使得es和kibana通信

sudo docker run --name es -d -p 9200:9200 -p 9300:9300 --network esnet -e "discovery.type=single-node" elasticsearch:6.8.8

12、启动docker出现的问题

Docker无法启动 error initializing graphdriver: driver not supported，解决办法：

sudo vim /etc/docker/daemon.json

重启

14、Linux命令行常用快捷键

  • Ctrl + U 剪切光标前的内容
  • Ctrl + K 剪切光标至行末的内容
  • Ctrl + Y 粘贴
  • Ctrl + E 移动光标至行末
  • Ctrl + A 移动光标至行首
  • Alt + F 跳向下一个空格
  • Alt + B 跳回上一个空格
  • Alt + Backspace 删除前一个单词
  • Ctrl + W 剪切光标后一个单词

17、Linux下查看硬件信息

a、命令行

安装dmidecode: sudo pacman -S dmidecode

使用：sudo dmidecode

查看内存信息：sudo dmidecode -t memory

具体用法：man dmidecode or dmidecode -h

b、图形化

安装cpu-g、hardinfo

18、Archlinux安装配置windows字体

拷贝C:\windows\fonts\下的所有字体

在ArchLinux中创建相应字体文件夹sudo mkdir /usr/share/fonts/WindowsFonts

将复制好的Windows字体复制进去sudo cp /window_fonts_path/fonts/* /usr/share/fonts/WindowsFonts

给字体文件夹权限sudo chmod 755 /usr/share/fonts/WindowsFonts/*

重新生成字体缓存fc-cahce -f
